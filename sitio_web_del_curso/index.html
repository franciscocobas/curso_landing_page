Comencemos el curso

Lo que hice fue hacer una Landing Page para presentarles el curso, este curso va a tratar 3 temas grandes. HTML, CSS y Bootstrap, algunos capaz que ya tienen conocimiento y otros no, por lo tanto vamos a organizarnos un poco diferente de cómo se organizan los cursos a los que todos estamos habituados. Por lo tanto nos vamos a guiar por 3 principios
sentiment_very_satisfied

Buena Onda!

Lo principal acá es pasarla bien. Tener buena onda y siempre responder con energía. Porque esto es algo que hacemos entre todos.

group

Colaboración

Vamos a trabajar de manera colaborativa. Pero, ¿Qué significa eso? Significa que los que no sabemos vamos a estar dispuestos a aprender de los que saben más y los que sabemos más vamos a tratar de ayudar a los que no tienen tanto conocimiento. Es una forma de ganar y ganar.

library_books

Aprender

La idea del curso es que todos nos llevemos algo, por lo tanto acá venimos a aprender entre todos, no se olviden que todos podemos hacerlo, capaz que somos muy buenos en algo y aprendemos enseñando al otro, y no nos creamos que no sabemos nada, a veces los que no sabemos ayudamos mucho a los que saben y es una forma de aprender.

Para escribir código vamos a utilizar el programa Visual Studio Code que es de código abierto y gratis. Si estamos acostumbrados a utilizar otro IDE también podemos hacerlo, adelante! Podemos apretar el icono de abajo para descargarlo.

Github

Vamos a utilizar un repositorio para este curso. Por lo tanto vamos a tener que saber como trabajar colaborativamente. Hacer Pull Request, hacer code review, buenas prácticas de programación y todo lo relacionado para ser un usuario intermedio-avanzado de GIT. Pueden acceder al repositorio acá mismo. Link

Bootstrap y Material Kit

Bootstrap es un kit de herramientas Open Source con HTML, CSS y Javascript. Se puede rápidamente prototipar tus ideas o construir tu aplicación con las variables Sass y mixins, Grid System responsive, extensibles componententes pre-construidos y poderosos plugins hechos en jQuery.
Material Kit es un kit premium de Bootstrap 4 que fue construido usando los principios de Material Design de Google.

Bootstrap

Vamos a utilizar Bootstrap 4 para nuestros proyectos. La documentación de ésta versión está en la página web oficial. Link

Material Kit

Vamos a utilizar Material Kit versión 2.0.4 para nuestros proyectos. La documentación de ésta versión está en la página web oficial. Link
